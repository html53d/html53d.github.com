You've probably noticed that a lot of websites lately have a featured area with content that slides or changes in some way. This is a great technique to show several pieces of content in a limited amount of space and a good way to engage the user. Featured 3D is a powerful plugin that can be integrated extremely easily in any Wordpress theme and can present your content in an unique and attractive manner in few mouse clicks.

  • Features
  • Dynamic options
  • Multiple sliders on a single site, each with unique set of options
  • Dynamic reflections and shadows
  • No Flash, just pure jQuery
  • Six color themes
  • Smooth 3D animation
  • Slides are HTML Content (can be anything)
  • Navigation controls are created dynamically
  • Ready to go in few mouse clicks
  • Many options to choose from
  • Dynamic reflections and shadows
  • Simple clean & valid markup
  • No coding required

3D motion control

  • Field of view

    You can select the field of view (viewing angle) of the camera from a predefined list

  • Move 3D stage horizontally & vertically

    You can give the viewers of your presentation to move the 3D stage in the two directions separately

Image enhancements

  • Image reflection

    You can add/remove a reflection for the main image of the slide

  • Image shadow

    You can also add/remove a shadow for the main image of the slide

Slide specific options

  • Dynamic background color

    A smooth transition of the background color from slide to slide is posible

  • Details location

    You can control the horizontal and vertical position of the text details for each of the slides independantly. Left, right, outside, top, bottom, middle, above, below... many configurations are possible.

  • Expose time

    You can set different expose times for each of slides independantly

Slider controls

  • Pager

    If checked, a pager control will be created which allows the user to switch manually from one slide to another

  • Buttons Next/Previuos

    You can choose if you want previous and/or next button controls

  • Button Play/Pause

    Alternative state stop/play button

  • Background color changer

    If checked, a pager container will allow you to switch manually from one slide to another

... and these are not all dynamic options. Each of the sliders in a page can be initiated with its own configuration set.
